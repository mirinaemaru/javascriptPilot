  • 함수는 객체(Object)이다. 변수로 할당, 파라미터로, 함수를 리턴할 수 있다.
  • 함수는 기본적인 빌딩 블럭
  • 하나의 함수는 하나의 기능만을 구현하도록 프로그래밍한다.
  • 함수의 이름은 커맨드형태, 동사의 형태로 한다.
  • 타입스크립트는 타입을 선언해야한다.
  • Object Parameter(객체 파라메터)는 레퍼런스를 전달한다.
  • Premitive Parameter 는 값을 전달한다.
  • Default Parameter(added in ES6)
  • 파라메터로 넘어오는 값이 없을 경우 미리 지정을 해 주면 기본 파라메터 값으로 변환된다.
  • Rest Parameter, ... 형태로 파라메터를 선언? 전달?한다. 그러면 배열형태로 전달된다.
  • Scope, Loacl Scope, Global Scope, 안에서는 밖의 변수가 보이고 밖에서는 안의 변수가 보이지 않는다.
